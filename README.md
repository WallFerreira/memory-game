# Memory-Game

Projeto Jogo da memória realizado a partir das aulas do Manual do Dev.

* Possui validador de caractéres mínimos no login
* validador de cartas iguais
* valida se todas as cartas já estão viradas
* Possui um time
* Nome do usuário
* Alerta quando ganha!!
